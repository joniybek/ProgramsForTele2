KNOWN ISSUES

- Cyrillic text is messed up in ODT files.
- To correctly load cyrillic text from old DOC files (Word 97-2003), use unicode font (Lucida Sans Unicode, etc.) with western european or central european character set.
- Umlauts in old DOC files (Word 97-2003) are replaced with closest ASCII equivalent. It does NOT affect search results.
- File names containing special characters (e.g. german umlauts) can not be opened.
